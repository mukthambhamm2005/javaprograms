# Java Programs

This repository contains Java programs created for learning and practicing core Java concepts. The programs cover a variety of topics, including basic syntax, object-oriented programming, control statements, loops, arrays, methods, and other fundamental concepts.

## Features

* Core Java examples
* Beginner-friendly programs
* Well-structured source code
* Practice exercises for Java learners

## Getting Started

### Prerequisites

* Java JDK 17 or later (or the version used for this project)
* IntelliJ IDEA or any Java IDE

### Clone the Repository

```bash
git clone https://github.com/mukthambhamm2005-jpg/javaprograms.git
```

### Run the Project

1. Open the project in IntelliJ IDEA.
2. Navigate to the `src` folder.
3. Open the Java file you want to run.
4. Click **Run** or execute the program from the terminal.
